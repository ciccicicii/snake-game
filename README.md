# 🐍 贪吃蛇大作战

一款可以在手机和电脑上玩的网页版贪吃蛇游戏。

## 🎮 如何玩

直接访问: **https://snake-game-three-psi-71.vercel.app**

或者本地运行:
1. 双击 `启动游戏.bat`
2. 浏览器打开 http://localhost:8080

## 📱 手机访问

确保手机和电脑在同一 WiFi 下，浏览器打开显示的 IP 地址即可。

## 🚀 部署到 Vercel

```bash
# 1. 安装 Vercel CLI
npm install -g vercel

# 2. 登录
vercel login

# 3. 部署
vercel --prod
```

## 🎯 游戏特性

- 移动端触控支持
- 键盘方向键控制
- 滑动手势控制
- 分数排行记录
- 难度递增
